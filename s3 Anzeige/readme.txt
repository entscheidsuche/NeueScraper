Ein paar Hilfsdateien, damit im Browser die Ansicht der Dateien auf S3 grendert werden können.
Dazu zählt die index.html, listdir.js und xml.css mit deren Hilfe eine Verzeichnisnavigation angezeigt wird.
Dann gibt es den Entscheid.xsl, welche die XML-Darstellung der Entscheide rendert.
Schliesslich die von den Spidern generierte Spiderliste.xml mit Spider.xsl, welche eine Übersicht über alle laufenden Spider gibt.
